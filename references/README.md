## How to Access References

### Peer-Reviewed Articles (Direct DOI Access)
Use the DOI link to access the paper directly through the publisher:

```
https://doi.org/[DOI]
```

Examples:
- Masanet 2020: https://doi.org/10.1126/science.aba3758
- Strubell 2019: https://doi.org/10.18653/v1/P19-1351
- Chien 2023: https://doi.org/10.1145/3606254
